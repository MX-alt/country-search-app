# 🌍 Country Search App

国名を入力すると、国旗と首都を表示するシンプルなWebアプリです。  
REST Countries API を利用してリアルタイムに国情報を取得します。

🔗 **Demo**  
https://mx-alt.github.io/country-search-app/

---

## 📌 概要

- 国名検索機能
- 国旗画像の表示
- 首都の表示
- エラー時のメッセージ表示

---

## 🛠 使用技術

- TypeScript
- Vite
- REST Countries API
- HTML / CSS
- Git / GitHub
- GitHub Pages（デプロイ）

---

## 🎯 制作目的

TypeScriptの基本文法、DOM操作、API通信、
